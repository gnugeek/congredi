congredi by gnugeek

Ruby interface to the New York Times Congress API.

= congredi * http://github.com/gnugeek/congredi/tree/master == DESCRIPTION: Ruby interface to the New York Times Congress API. ( http://developer.nytimes.com/docs/congress_api ) == FEATURES/PROBLEMS: * Fetch roll-call votes by congress, chamber, session, and roll-call * Fetch members of congress by congress and chamber * Fetch voting record by member of congress * Fetch biographical data by member of congress == SYNOPSIS: require 'congredi' true m = Congredi::Members.new({:congress => 111, :chamber => 'senate'}) member = m.members.first member.name "Daniel Akaka" member.member_id "A000069" bio = Congredi::Bio.new({:member_id => member.member_id}) bio.date_of_birth "1924-09-11" votes = Congredi::Votes.new({:member_id => member.member_id}) vote = votes.votes.first vote.position "Yes" vote.roll_call "215" rc = Congredi::RollCall.new({:congress => vote.congress, :chamber => vote.chamber, :session => vote.session, :rollCall => vote.roll_call}) rc.bill_number "H.R.7005" rc.description "Motion to Invoke Cloture on the Motion to Proceed to Consider H.R. 7005.; Alternative Minimum Tax Relief Act of 2008" rc.democratic_majority_position "Yes" rc.republican_yes "10"

Dependencies

hpricot

Install

* gem build congredi.gemspec * sudo gem install congredi-X.X.X.gem * edit lib/congressapi/congredi.rb and add in your API key
